# RC-Car
Машинка с Arduino на борту, управляется с телефона по Bluetooth.

Используемые пины:
* 0 - не используется
* 1 - не используется
* 2 - Tx HC-06
* 3 - Rx HC-06
* 4 - не используется
* 5 - IN1 Н-моста (полярность выхода 1)
* 6 - IN2 Н-моста (напряжение выхода 1)
* 7 - Управляющий провод сервопривода
* 8 - не используется
* 9 - не используется
* 10 - IN3 Н-моста (полярность выхода 2)
* 11 - IN4 Н-моста (напряжение выхода 2)
* 12 - Фары
* 13 - не используется
* A0 - не используется
* A1 - не используется
* A2 - не используется
* A3 - не используется
* A4 - не используется
* A5 - не используется
